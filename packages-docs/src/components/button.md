## Button API

### Button Attributes

| 属性名             | 说明               | 类型              | Default           |
| ----------------- | ----------------- | ----------------- | ----------------- |
| label | 按钮内容 | string | — |
| style | 内连样式 | string | — |
| class | 类名 | string | — |
| type  | 按钮类型 | `text`、`link` | — |
| shape | 按钮形状 | `plain`、`round`、`circle`、`secondary` | — |
| theme | 按钮形状 | `primary`、`success`、`warning`、`danger`、`info` | — |
