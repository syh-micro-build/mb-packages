## Button API

### Button Attributes

| 属性名             | 说明               | 类型              | Default           |
| ----------------- | ----------------- | ----------------- | ----------------- |
| label | 按钮内容 | string | — |
| color | 按钮颜色 | `primary`、`success`、`warning`、`danger`、`info`、`default` | — |
| underline | 是否下划线 | boolean | — |
| disabled | 是否禁止点击 | boolean | — |
| href | 跳转链接 | string | — |
| target | 原生属性 | `_blank`、`_parent`、`_self`、`_top` | — |
| icon | 自定义 Icon | HTMLElement、Object | — |
| loading | 是否加载 | string | — |
